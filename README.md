# Editor Excel → CRM Mejorado

## 🚀 Mejoras Implementadas
- **Multi-hoja**: Selector de hoja por archivo.
- **Edición inline**: Preview editable con st-aggrid o data_editor.
- **Refactor modular**: config.py, utils.py.
- **Estado**: Multi-archivo, perfiles mapeo persistentes.

## 📦 Instalación
```
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Características
- Mapeo columnas origen → 26 CRM fijas.
- Deduplicación inteligente por COD_CREDITO.
- Stats/Validaciones automáticas.
- Export XLSX/ZIP individual/combinado.

## 📝 Uso
1. Subir .xlsx (multi).
2. Seleccionar archivo/hoja.
3. Mapeo (auto o manual, guardar perfil).
4. Preview/edit → Exportar.

**¡App lista y mejorada!** 🎉
